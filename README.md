# Docker for sites based on Joomla
 
## What is inside

* MySQL 
* Joomla
* PhpMyadmin
 
## Workflow

* install docker and docker-compose (ensure that you can run docker without sudo)
* cd docker
* ./run.sh -h read help 
* ./run.sh -s start services
* ./run.sh -z create database dump

www - src folder for your Joomla site
